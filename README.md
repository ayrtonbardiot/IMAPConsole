<h2>
My mail hoster is not in hoster.txt ...
</h2>
No problem, you can add it in hoster.txt with format : MAIL_DOMAIN:IMAP_SERVER:PORT
Exemple : gmail.com:imap.gmail.com:993

You can make a PR for add it to the source ;)
